# How to debug in vs code with arguments! 

Steps: 
1) You go to the Run and Debug page (Or press Ctrl + Shift + D)
2) Click the "xreate a launch.json file" below the Run & Debug button.
3) In the .json file that opens do something like this:

         {
              // Use IntelliSense to learn about possible attributes.
              // Hover to view descriptions of existing attributes.
              // For more information, visit: https://go.microsoft.com/fwlink/?linkid=830387
              "version": "0.2.0",
              "configurations": [          
                  {
                      "name": "Python Debugger: testing.py",
                      "type": "debugpy",
                      "request": "launch",
                      "program": "testing.py",
                      "args": [
                          "--root", "/ssd/path_to_dataset",
                          "--save_dir", "runs/yolox",
                          "--out_dir", "./filtered_test_eval",
                          "--epoch", "10",
                          "--use_ema",
                          "--img_w", "304", "--img_h", "240",
                          "--input_size", "640", "640",
                          "--history_chunks", "20",
                          "--micro_slices", "1",
                          "--workers", "0",
                          "--conf_thre", "0.01",
                          "--nms_thre", "0.65",
                          "--test_max_files_per_folder", "-1"
                      ],
                      "console": "integratedTerminal"
                  }
              ]
          }

4) Go to the Run and Debug page and at the top find the "Python Debugger: testing.py" next to a green arrow. Press the arrow andf it will Debug.
5) If you want to check a breaking point in a site-package, set:

       "justMyCode": false
         under the "console" in launch.json so breakpoints in site-packages are honored.
